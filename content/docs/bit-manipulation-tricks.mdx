## ðŸ”¹ 4. Checking if a Number is a Power of Two

```javascript
function isPowerOfTwo(n) {
  return n > 0 && (n & (n - 1)) === 0;
}

console.log(isPowerOfTwo(8));  // true
console.log(isPowerOfTwo(10)); // false
```

âœ… **A power of two has only one `1` bit in binary.**

## ðŸ”¹ 5. Counting Set Bits (Hamming Weight)

```javascript
function countSetBits(n) {
  let count = 0;
  while (n > 0) {
    count += n & 1;
    n >>= 1;
  }
  return count;
}

console.log(countSetBits(9)); // 2 (1001)
```

âœ… **Counts the number of `1` bits in a number.**

## ðŸ”¹ 6. Finding the Only Non-Duplicate Number

Given an array where every number appears twice except for **one**, find that number.

```javascript
function findUnique(arr) {
  return arr.reduce((acc, num) => acc ^ num, 0);
}

console.log(findUnique([2, 3, 2, 4, 4])); // 3
```

âœ… **Uses XOR properties (`x ^ x = 0`) to isolate the unique number.**

## ðŸ”¹ 7. Reversing Bits

```javascript
function reverseBits(n) {
  let result = 0;
  for (let i = 0; i < 32; i++) {
    result = (result << 1) | (n & 1);
    n >>= 1;
  }
  return result >>> 0;
}

console.log(reverseBits(43261596)); // 964176192
```

âœ… **Shifts bits and reconstructs the reversed number.**
