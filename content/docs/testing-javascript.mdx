## ðŸ”¹ 3. Integration Testing with Jest & Supertest

### **3.1 Install Supertest for API Testing**
```bash
npm install --save-dev supertest express
```

### **3.2 Writing an Express API Test**

ðŸ“Œ **`server.js` (Simple API)**
```javascript
const express = require("express");
const app = express();

app.get("/hello", (req, res) => res.json({ message: "Hello, World!" }));

module.exports = app;
```

ðŸ“Œ **`server.test.js` (Integration Test)**
```javascript
const request = require("supertest");
const app = require("./server");

test("GET /hello should return message", async () => {
  const res = await request(app).get("/hello");
  expect(res.statusCode).toBe(200);
  expect(res.body.message).toBe("Hello, World!");
});
```

âœ… **Integration tests check API responses without running the full app.**
