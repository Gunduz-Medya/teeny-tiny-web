## ðŸ“Œ Introduction to React

React is a popular JavaScript library for building user interfaces, developed by Facebook. It allows developers to create reusable UI components and manage state efficiently.

### ðŸ”¹ Setting Up a React Project

You can create a new React project using **Create React App (CRA)** or **Vite**.

#### Using Create React App:
```bash
npx create-react-app my-app
cd my-app
npm start
```

#### Using Vite (Faster alternative):
```bash
npm create vite@latest my-app --template react
cd my-app
npm install
npm run dev
```

### ðŸ”¹ JSX - JavaScript XML

JSX allows writing HTML-like syntax within JavaScript.

```jsx
const element = <h1>Hello, React!</h1>;
```

JSX must be wrapped in a parent element:

```jsx
return (
  <div>
    <h1>Welcome</h1>
    <p>This is a React component.</p>
  </div>
);
```

### ðŸ”¹ Components in React

React apps are built using components. There are two main types:

#### Functional Component:
```jsx
function Greeting() {
  return <h1>Hello, World!</h1>;
}
```

#### Class Component:
```jsx
class Greeting extends React.Component {
  render() {
    return <h1>Hello, World!</h1>;
  }
}
```

### ðŸ”¹ Props - Passing Data

Props (short for properties) allow passing data between components.

```jsx
function Welcome(props) {
  return <h1>Hello, {props.name}!</h1>;
}

// Usage:
<Welcome name="Alice" />
```

### ðŸ”¹ State - Managing Data

State is used to manage dynamic data inside components.

```jsx
import { useState } from "react";

function Counter() {
  const [count, setCount] = useState(0);

  return (
    <div>
      <p>Count: {count}</p>
      <button onClick={() => setCount(count + 1)}>Increment</button>
    </div>
  );
}
```

### ðŸ”¹ Handling Events

React allows event handling similar to HTML but uses camelCase syntax.

```jsx
function Button() {
  function handleClick() {
    alert("Button clicked!");
  }

  return <button onClick={handleClick}>Click Me</button>;
}
```

### ðŸ”¹ Conditional Rendering

You can render components conditionally using the `if` statement or ternary operators.

```jsx
function Message({ isLoggedIn }) {
  return isLoggedIn ? <h1>Welcome back!</h1> : <h1>Please sign in</h1>;
}
```

### ðŸ”¹ Lists and Keys

Rendering lists in React requires using `.map()` and assigning unique `key` props.

```jsx
const users = ["Alice", "Bob", "Charlie"];

function UserList() {
  return (
    <ul>
      {users.map((user, index) => (
        <li key={index}>{user}</li>
      ))}
    </ul>
  );
}
```

### ðŸ”¹ UseEffect Hook (Handling Side Effects)

`useEffect` runs side effects such as data fetching, subscriptions, or manual DOM updates.

```jsx
import { useState, useEffect } from "react";

function Timer() {
  const [time, setTime] = useState(0);

  useEffect(() => {
    const interval = setInterval(() => {
      setTime((prev) => prev + 1);
    }, 1000);

    return () => clearInterval(interval); // Cleanup
  }, []);

  return <p>Timer: {time} seconds</p>;
}
```

### ðŸ”¹ Fetching Data with Fetch API

Fetching data from an API in React.

```jsx
import { useState, useEffect } from "react";

function FetchData() {
  const [data, setData] = useState([]);

  useEffect(() => {
    fetch("https://jsonplaceholder.typicode.com/posts")
      .then((response) => response.json())
      .then((json) => setData(json));
  }, []);

  return (
    <ul>
      {data.map((post) => (
        <li key={post.id}>{post.title}</li>
      ))}
    </ul>
  );
}
```
